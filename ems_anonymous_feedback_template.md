# 광주·전남 구급대원 무기명 현장 제보 설문 템플릿

## 운영 원칙

- 이름, 연락처, 소속 센터명, 차량번호, 정확한 출동 주소는 수집하지 않는다.
- 이메일 수집, 로그인 요구, 응답자 IP 수집 옵션을 끈다.
- 공개 시에는 개인·기관 식별 가능성을 낮춘 집계와 익명화 사례만 사용한다.
- 특정 개인 비난보다 이송·수용·전원 체계의 병목을 파악하는 데 목적을 둔다.
